RCT Treatment Interface

Static deployment of the experimental UI used inside a Qualtrics survey. The Qualtrics question loads embed.html in an iframe — that is the only file participants see.

Local simulators (developer tools)

  • index-llm.html — Qualtrics-side simulator for the LLM condition. Embeds the iframe and mimics the postMessage bridge.
  • index-search.html — same, for the SEARCH condition.

Direct embed (what Qualtrics loads)

  • embed.html?condition=LLM&pid=demo
  • embed.html?condition=SEARCH&pid=demo
For Qualtrics, paste an <iframe> into the question Question Text pointing at YOUR_DEPLOYED_URL/embed.html?condition=LLM&pid=${e://Field/ResponseID}, and paste qualtrics-question-js.js into that question's JavaScript panel. See the README for the full setup.
